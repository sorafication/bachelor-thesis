Integration von Amazon Cognito in Web- und mobile Apps

PDF

Wenn neue Benutzer Ihre App kennenlernen oder wenn bestehende Benutzer sie wieder verwenden, müssen sie sich zuerst registrieren oder anmelden. Durch die Integration von Amazon Cognito in Ihrem Client-Code verbinden Sie Ihre App mit der AWS-Backend-Funktionalität, die Authentifizierungs- und Autorisierungsworkflows unterstützt. Ihre App verwendet die Amazon Cognito-API beispielsweise, um neue Benutzer in Ihrem Benutzerpool zu erstellen, Benutzerpool-Token abzurufen und temporäre Anmeldeinformationen aus Ihrem Identitäten-Pool zu erhalten. Verwenden Sie die SDKs und Bibliotheken, die das AWS Amplify-Framework bereitstellt, um Amazon Cognito in Ihre Web- und mobile App zu integrieren.

Amazon Cognito-Authentifizierung mit dem AWS Amplify-Framework

AWS Amplify stellt Services und Bibliotheken für Entwickler und Web- und mobilen Apps bereit. Mit AWS Amplify können Sie Apps erstellen, die in Backend-Umgebungen aus AWS-Services integriert werden können. Für die Bereitstellung Ihrer Backend-Umgebung und die Integration von AWS-Services in Ihren Client-Code verwenden Sie das AWS Amplify-Framework. Das Framework stellt eine interaktive Befehlszeilenschnittstelle (Command Line Interface, CLI) bereit, die Sie bei der Konfiguration von AWS-Ressourcen für in Kategorien unterteilte Funktionen unterstützen. Diese Kategorien umfassen u. a. Analyse, Speicherung und Authentifizierung. Das Framework stellt darüber hinaus allgemeine SDKs und Bibliotheken für Web- und mobile Plattformen bereit, einschließlich iOS, Android und JavaScript. Zu den unterstützten JavaScript-Frameworks gehören React, React Native, Angular, Ionic und Vue. Jedes SDK und jede Bibliotheken umfasst Authentifizierungsoperationen, mit denen Sie die Authentifizierungsworkflows implementieren können, die Amazon Cognito unterstützt.

Informationen zur Verwendung des AWS Amplify-Frameworks zum Hinzufügen der Authentifizierungsfunktion zu Ihrer App finden Sie in der AWS Amplify-Dokumentation für Ihre Plattform:

  • AWS Amplify-Authentifizierung für JavaScript

  • AWS Amplify-Authentifizierung für iOS

  • AWS Amplify-Authentifizierung für Android
